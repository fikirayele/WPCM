# **App Name**: FaithConnect

## Core Features:

- Consultation Request: Students can submit consultation requests with details like department, problem description, and preferred time.
- Admin Consultant Assignment: Admin assigns consultants to pending requests based on the selected department.
- Real-time Chat: Secure, real-time chat between students and assigned consultants.
- Consultation Status Management: System manages consultation statuses (PENDING, ASSIGNED, ACTIVE, PAUSED, COMPLETED).
- Admin Dashboard: Comprehensive dashboard for user, department, consultant, and consultation management.
- Public Website: Public-facing website with church information, news, announcements, and donation options.
- Content Summarizer: Generative AI tool that provides summaries of consultation chat histories to aid consultants.

## Style Guidelines:

- Primary color: Deep purple (#4F3A65) to evoke spirituality and trust.
- Background color: Soft lavender (#E6E0EB), a very light tint of the primary.
- Accent color: Muted teal (#70A4A2) for highlights and calls to action.
- Headline font: 'Playfair', serif, for titles, announcements, and headlines; body font: 'PT Sans', sans-serif, for main text content and body.
- Use consistent, professional icons for navigation and features. Consider line icons for a clean and modern look.
- Clean and intuitive layout with clear sections for each feature. Prioritize ease of navigation.
- Subtle transitions and animations to enhance user experience and provide feedback on interactions.
